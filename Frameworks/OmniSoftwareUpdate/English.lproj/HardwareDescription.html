When ${APP} checks for new and updated versions of itself, it optionally includes some basic information about your system configuration. We use this data to determine what systems our customers are using and therefore what systems are most important to support.

Omni will never release information about an individual's computer configuration â€” only statistical information about all collected configurations. We honestly respect your privacy.

If you prefer not to submit your system info, no problem - simply do not turn on the checkbox in the Omni Software Update preferences pane in this application.

We do make the statistics we gather public so that other developers can also benefit from this knowledge. You can see what the information we release looks like by viewing the Omni Software Update page.

The following pieces of information are always sent with the query:

OmniSoftwareUpdate version ${OSU_VER}
Application ID ${OSU_APP_ID}
Application Version ${OSU_APP_VER}
Update Track ${OSU_TRACK}
Visible Update Tracks ${OSU_VISIBLE_TRACKS}

If enabled in the 'Update' preferences, the following additional information is sent with the query:

${OTHERVARS}
UUID ${uuid}
A random string that anonymously identifies your computer. This allows us to keep our database up to date when your system configuration changes. This is not associated with your name or any other personal information.
License Type ${license-type}
Your license key itself will never be sent.
OS Version ${os}
Preferred Language ${lang}
General Location Latitude: ${LATITUDE-DMS} Longitude: ${LONGITUDE-DMS}
This general location is based on the closest major city that the System Preferences application selected when you picked a time zone while setting up your computer. You can see exactly what location this reports at MapQuest.
Computer Model ${machine_name} (${hw-model})
